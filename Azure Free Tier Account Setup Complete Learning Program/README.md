# Summary Report

Region Selection: West US (westus)

I have selected the West US.This region is part of Microsoft Azure’s global infrastructure and hosts data centers located in the western United States.

This region was chosen because 
Azure services
- Low latency for users located in or near the western United States
- Support for high availability and redundancy features
- Compliance with U.S. data residency and regulatory requirements
- Cost-effective option compared to some other premium regions

---

Shared Responsibility Model defines how security and management responsibilities are divided between Microsoft (cloud provider) and the customer.

 However, security is a shared responsibility. Microsoft secures the underlying infrastructure, while the customer is responsible for securing configurations, data, and applications running in the cloud.
